# كيف تحصل على رسالة الخطأ الكاملة من Netlify

لو الـ Deploy لسه فاشل، نحتاج **سطر الخطأ بالظبط** من الـ log عشان نصلح السبب.

## الخطوات

1. ادخل على **https://app.netlify.com** وافتح موقعك (Stella).
2. من القائمة الجانبية اضغط **Deploys** (أو **النشر**).
3. اضغط على الـ Deploy اللي عليه **Failed** (أحمر).
4. في الصفحة الجديدة اضغط **Deploy log** أو **View build log** أو السهم (>) بجانب الـ Deploy.
5. الـ log يفتح — **انزل لآخر الصفحة** (أسفل الـ log).
6. ابحث عن أول سطر فيه كلمة **Error** أو **error** أو **Failed** (غالباً بلون أحمر أو مميز).
7. **انسخ السطر كامل** أو التقط **صورة للشاشة** من أول ظهور كلمة Error لحد نهاية السطر (أو 3–4 أسطر بعدها).
8. أرسل النص أو الصورة هنا.

## مثال

مثلاً لو ظهر شيء زي:

```
Error: Cannot find module 'xyz'
```

أو:

```
TypeError: xxx is not a function
```

انسخه كامل وأرسله — هنساعدك نحدد السبب ونصلحه.
