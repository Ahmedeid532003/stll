# Stella — موقع مستلزمات حريمى

[![Netlify Status](https://api.netlify.com/api/v1/badges/e007df0f-e7ea-4ff9-ba31-b91ebac0b83c/deploy-status)](https://app.netlify.com/projects/stella0/deploys)

موقع لعرض منتجات الملابس الحريمى مع لوحة تحكم للأدمن وإدارة مخزون.

## التشغيل

```bash
npm install
npm run dev
```

ثم افتح [http://localhost:3000](http://localhost:3000).

## لوحة الأدمن

- الرابط: `/admin`
- كلمة المرور الافتراضية: **yomna2003**

من لوحة الأدمن يمكنك:
- **المنتجات**: إضافة منتجات (اسم، وصف، سعر، صور، مقاسات، ألوان) وإخفاء منتج من العرض
- **المخزون**: إضافة وتعديل الكميات (يظهر للأدمن فقط، وينقص تلقائياً عند كل طلب)
- **الطلبات**: عرض الطلبات وتحديث حالتها (قيد الانتظار، مؤكد، تم الشحن، تم التوصيل)

## الصور

عند إضافة منتج، الصور تُدخل كروابط (URL). يمكنك رفع الصور على أي استضافة صور ثم لصق الرابط في حقل "روابط الصور" (سطر لكل صورة أو مفصولة بفاصلة).

## البيانات

البيانات تُحفظ محلياً في مجلد `data/` (ملفات JSON). للمشروع الحقيقي يُفضّل استخدام قاعدة بيانات.
